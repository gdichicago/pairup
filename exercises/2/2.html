Level 2: Mad Libs

Using JSBin, set up a screen where you're viewing the HTML, JavaScript, and Output.

You will need to create the following elements in your HTML:

  • A header
  • An unordered list with 3 items. Each item should contain a label (noun, adjective, or someone's name) and a text box for the user to type in.
  • A button
  • An empty div with the id story

Create a new function with an event listener for the button on your page.

In the function, retrieve the current values of the form input elements, make a story from them, and output that in the story div (like "Pamela really likes pink cucumbers.").

See solution at your own peril!
