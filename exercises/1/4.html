Level 1: Media queries

The goal of this exercise is to use CSS media queries to make a webpage appear different for printing and for different window sizes.

  • Create a new page with some HTML and basic CSS. You'll need a header, navigation menu, content, sidebar, and a footer. Try and really get some content in there so you have lots of stuff to play with.
  • Add a CSS media query and appropriate styles so the website adjusts for smaller screens. Start with one size - like a 13" laptop or an iPhone. Google to get the screen resolution in pixels.
  • Some things to try:
    • Hide the header and footer.
    • Change the way your navigation is displayed.
    • Show your sidebar underneath your main content.
    • Remove the excess padding from the body.
    • Fix the header at the top of the page no matter how far down you scroll.
    • Change the font size or font family.
  • Try adding in another media query for a different size.
